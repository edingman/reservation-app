Step-by-step Google Setup Guide

1. Create a Google Cloud Project

  1. Go to console.cloud.google.com
  2. Click the project dropdown at the top → New Project
  3. Name it (e.g. "Room Booking") and click Create
  4. Make sure the new project is selected in the dropdown

2. Enable the Required APIs

  1. Go to APIs & Services → Library (left menu)
  2. Search for and enable each of these:
    • Google Calendar API
    • Admin SDK API
    • Google Drive API
  3. Click Enable on each one

3. Create a Service Account

  1. Go to APIs & Services → Credentials
  2. Click + Create Credentials → Service Account
  3. Give it a name (e.g. "room-booking-service")
  4. Skip the optional role/access steps and click Done
  5. Click on the newly created service account
  6. Go to the Keys tab
  7. Click Add Key → Create New Key → JSON → Create
  8. A JSON file will download — this is your service account key

4. Enable Domain-Wide Delegation

  1. On the service account page, click Show Advanced Settings or look under Domain-wide delegation
  2. Check Enable Google Workspace Domain-wide Delegation
  3. Save and copy the Client ID (a long number)

5. Grant API Scopes in Google Admin

  1. Go to admin.google.com
  2. Navigate to Security → Access and data control → API controls
  3. Click Manage Domain Wide Delegation
  4. Click Add new
  5. Paste the Client ID from step 4
  6. In the OAuth scopes field, paste all three scopes (comma-separated): https://www.googleapis.com/auth/calendar,https://www.googleapis.com/auth/admin.directory.resource.calendar.readonly,https://www.googleapis.com/auth/drive.file
  7. Click Authorize

6. Configure in This App

  1. Upload the JSON key file using the upload area above
  2. Enter the Delegated User Email — this must be a Google Workspace admin email (e.g. admin@yourdomain.com)
  3. Leave Customer ID as "my_customer" (default works for most setups)
  4. Click Save Settings then Test Connection
  5. If connected, your Google Workspace room resources will appear below
  6. Go to the Rooms tab and link each room to its Google resource
  7. Enable Auto-backup in the Data Backup section below to protect your data
